# @figkit/patterns

Composed UI patterns built on top of @figkit/ui components.

## Installation

```bash
npm install @figkit/patterns
# or
pnpm add @figkit/patterns
# or
yarn add @figkit/patterns
```

## Peer Dependencies

This package requires:
- `@figkit/ui` - UI component library
- `react` and `react-dom` (v18+ or v19+)

## Usage

```tsx
import { PatternName } from '@figkit/patterns';
import '@figkit/patterns/index.css';
```

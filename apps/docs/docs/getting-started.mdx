# Getting Started

Welcome to the FigKit Design System! This guide will help you get up and running quickly.

## Installation

Install the FigKit UI package from npm:

```bash
npm install @figkit/ui @figkit/themes
```

Or with pnpm:

```bash
pnpm add @figkit/ui @figkit/themes
```

## Basic Setup

Import the theme CSS and start using components:

```tsx
import '@figkit/themes/dist/tokens.css';
import '@figkit/ui/dist/index.css';
import { Button } from '@figkit/ui';

function App() {
  return (
    <div data-brand="default" data-theme="light">
      <Button variant="primary">Hello FigKit</Button>
    </div>
  );
}
```

## Theming

FigKit supports multiple brands and themes. Set the brand and theme using data attributes:

```tsx
<div data-brand="default" data-theme="light">
  {/* Your app */}
</div>
```

Available brands: `default`, `figjam`, `devmode`, `slides`
Available themes: `light`, `dark`
